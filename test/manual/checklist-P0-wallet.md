# 人工验收清单: ACH-USER-001 Secure Digital Wallet

> 本清单用于人工验收无法通过自动化测试验证的用户体验项
> 验收人员: QA/产品/开发
> 预估时间: 30分钟

---

## 验收项 1: 钱包创建体验 (< 60秒)

### 测试步骤
1. 清除浏览器缓存或打开无痕模式
2. 访问应用首页
3. 点击"创建钱包"按钮
4. 记录从点击到钱包创建完成的总时间
5. 验证钱包地址已显示在界面上

### 期望结果
- [ ] 整个过程在60秒内完成
- [ ] 钱包地址正确显示
- [ ] 有明确的进度提示

### 通过标准
- 实际用时 ≤ 60秒: ✅ 通过
- 实际用时 > 60秒: ❌ 不通过

---

## 验收项 2: Keplr 钱包集成体验

### 前置条件
- 已安装 Keplr 浏览器扩展
- 已创建或导入 Keplr 钱包

### 测试步骤
1. 点击"连接 Keplr"按钮
2. 在 Keplr 弹窗中确认连接
3. 验证应用显示的钱包地址与 Keplr 一致
4. 尝试通过 Keplr 签名一笔交易

### 期望结果
- [ ] Keplr 弹窗正常弹出
- [ ] 应用正确显示 Keplr 钱包地址
- [ ] 交易签名流程顺畅
- [ ] 签名后交易成功广播

### 通过标准
- 所有检查项通过: ✅ 通过
- 任一检查项失败: ❌ 不通过

---

## 验收项 3: 手机钱包集成体验

### 前置条件
- 已安装手机钱包应用 (Keplr Mobile / Trust Wallet)

### 测试步骤
1. 在电脑上点击"连接手机钱包"
2. 使用手机扫描二维码
3. 在手机钱包中确认连接
4. 验证电脑应用显示的钱包地址
5. 尝试从手机钱包签名交易

### 期望结果
- [ ] QR 码正常显示
- [ ] 手机扫描成功
- [ ] 电脑应用正确显示手机钱包地址
- [ ] 手机签名流程顺畅

### 通过标准
- 所有检查项通过: ✅ 通过
- 任一检查项失败: ❌ 不通过

---

## 验收项 4: 私钥导出安全性体验

### 测试步骤
1. 进入钱包设置页面
2. 点击"导出私钥"
3. 验证需要输入密码/验证身份
4. 查看私钥显示方式
5. 验证有安全提示

### 期望结果
- [ ] 导出前需要身份验证
- [ ] 私钥以安全方式显示（如部分隐藏）
- [ ] 有明确的安全警告提示
- [ ] 提供复制功能
- [ ] 一段时间后自动隐藏

### 通过标准
- 所有检查项通过: ✅ 通过
- 缺少任一安全措施: ⚠️ 需改进
