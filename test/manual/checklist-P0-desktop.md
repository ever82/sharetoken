## 验收项 1: Windows 平台开箱即用

### 前置条件
- Windows 10/11 系统
- 无 Node.js / npm / Go 环境
- 无 Keplr 浏览器扩展

### 测试步骤
1. 从 Releases 下载 `ShareToken-Desktop-Windows.zip`
2. 解压到任意目录
3. 双击 `ShareToken.exe`
4. 观察启动过程

### 期望结果
- [ ] 下载文件正常
- [ ] 解压过程顺利
- [ ] 双击后应用启动
- [ ] 启动时间 < 30秒
- [ ] 无需安装任何依赖
- [ ] 无需运行命令行

- 所有检查项通过: ✅ 通过
- 任一检查项失败: ❌ 不通过
