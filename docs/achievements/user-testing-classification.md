# ShareToken 用户成就验收方式分类

> 基于 `docs/achievements/for-user.md` 分析
> 生成日期: 2026-03-11

---

## 分类标准

### 可自动化测试 (Automated Testing)
- ✅ 可通过代码/脚本验证的功能
- ✅ 有明确的数值指标（时间、准确率、百分比等）
- ✅ 后端API、合约逻辑、数据处理
- ✅ 性能指标（TPS、延迟、响应时间）

### 需人工验收 (Manual Testing)
- 👤 需要主观体验和判断
- 👤 UI/UX视觉和交互体验
- 👤 跨浏览器/设备兼容性体验
- 👤 自然语言交互质量
- 👤 安装和配置流程体验
