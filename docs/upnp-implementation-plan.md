# UPnP 完整实现计划

## 现状
当前 `app/network/nat.go` 中的 UPnP 实现是占位符，仅记录日志，没有实际功能。

## 实现方案

### 1. 添加依赖
```bash
go get github.com/huin/goupnp
```

### 2. 修改 setupUPnP() 函数
使用 goupnp 库实现：
- SSDP 发现 IGD 设备
- 获取外部 IP
- 添加端口映射
- 启动保活机制

### 3. 测试要求
- 需要真实路由器支持 UPnP
- 需要公网 IP 环境
- 需要验证端口是否正确映射

## 验收标准
- [ ] 节点启动时自动发现 UPnP 路由器
- [ ] 自动将 P2P 端口 (26656) 映射到外部
- [ ] 获取外部 IP 地址
- [ ] 保活机制维持映射不超时
- [ ] 节点停止时清理映射

## 预估工作量
- 开发：4-6 小时
- 测试：需真实网络环境，时间不定
